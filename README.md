## Comment l'utiliser ?
- Autoriser l'accès au microphone.
- Maintenir enfoncée la touche d'enregistrement vocal pendant toute la durée de l'enregistrement.
- Relâcher la touche pour terminer l'enregistrement.
- Attendre la réponse.

# Techno 
- NextJS 14
- MongoDB (chatHistory)
- serverAction
- ai/react
- STT / TTS avec openAI
